## 一、先搞清楚：什么是 IPv6，我需要它吗？

IPv6 是下一代互联网协议地址格式，相比 IPv4 的 32 位地址，IPv6 有 128 位，地址数量几乎无限。

对 VPS 用户来说，用上 IPv6 有几个现实好处：

**IPv4 被封后的备用通道。** 搬瓦工 VPS 的 IPv4 地址有时候会被屏蔽，而 IPv6 目前相对比较干净，可以作为 IPv4 不可用时的备用访问方式。

**访问 IPv6-Only 资源。** 部分资源或服务只提供 IPv6 地址，有了 IPv6 才能直接访问。

**双栈支持更灵活。** 同时拥有 IPv4 和 IPv6，你的服务器在做网络配置、内容分发等场景下会更灵活。

当然，如果你只是拿 VPS 跑个网站、搭个代理，IPv6 不是必须的，有了更好，没有也无所谓。

## 三、IPv6 的基本概念（新手扫盲）

在进入操作之前，简单解释几个容易混淆的概念：

**IPv4 vs IPv6：** IPv4 地址长这样 `192.168.1.1`，IPv6 地址长这样 `2001:0db8:85a3::8a2e:0370:7334`。后者更长，也更多。

**双栈（Dual Stack）：** 指服务器同时拥有 IPv4 和 IPv6 地址，两个协议都能用。搬瓦工启用 IPv6 后，你的 VPS 就是双栈的。

**KiwiVM：** 搬瓦工自研的 VPS 控制面板，你购买之后通过这个面板管理服务器——开机关机、重装系统、切换机房、启用 IPv6，都在这里完成。

**CN2 GIA：** 中国电信的高端直连线路，延迟低、速度快，是搬瓦工在国内受欢迎的核心原因之一。目前 IPv6 暂不走这条线路。

## 五、如何在 KiwiVM 启用 IPv6（分步教程）

**第一步：确认你的套餐和机房支持 IPv6**

登录 KiwiVM 控制面板（从搬瓦工后台客户区进入），查看当前机房。KiwiVM 的机房迁移页面里，支持 IPv6 的机房旁边会标注 `IPv6` 字样，非常直观。

如果你当前的机房不支持 IPv6，可以在 KiwiVM 里免费迁移到支持的机房（注意迁移会更换 IP 地址）。

**第二步：在 KiwiVM 里分配 IPv6 地址**

进入 KiwiVM 控制面板后，找到 **IPv6** 相关选项（通常在左侧菜单），点击启用/分配 IPv6 地址。系统会自动为你的 VPS 分配一个 IPv6 地址。

**第三步：重启 VPS**

这一步很多人会漏掉。分配完 IPv6 地址之后，**必须重启 VPS 才能生效**，不然 IPv6 不会激活。

**第四步：验证是否成功**

SSH 登录到 VPS，运行命令：

bash
ifconfig


找到有 `inet6` 字样的行，后面跟的就是你的 IPv6 地址。

然后可以测试连通性：

bash
ping6 google.com


如果能 ping 通，说明 IPv6 已经正常工作了。

如果 ping 不通，可以检查一下 `/etc/network/interfaces` 文件里的 IPv6 配置是否正确，或者参考 KiwiVM 面板里的 Assistant Amy 智能助手帮你排查问题。

## 六、新手常见问题

**Q：我买的 CN2 GIA-E 套餐，但当前机房不在支持列表里，能用 IPv6 吗？**

不能。需要先迁移到支持 IPv6 的机房（如 DC6、DC9 等），再启用。迁移免费，在 KiwiVM 里操作即可，迁移后 IP 会变化。

**Q：IPv6 开了之后，IPv4 还能用吗？**

当然可以。开启 IPv6 不影响 IPv4，两个地址并存，这就是双栈。

**Q：香港套餐支持 IPv6 吗？**

目前香港套餐不在 IPv6 支持列表里。搬瓦工 IPv6 目前覆盖的是美国西海岸机房，香港、日本等高端机房暂未支持。

**Q：搬瓦工 IPv6 的速度和延迟怎么样？**

现阶段 IPv6 走的是普通线路，没有 CN2 GIA 优化，所以对国内用户而言延迟会比 IPv4 高，速度也不如优化线路。如果主要使用场景对延迟敏感，当前阶段不建议主要依赖 IPv6。

**Q：有优惠码可以用吗？**

目前最新可用的优惠码是 **NODESEEK2026**，全场常规套餐可用，循环折扣约 6.77%。该优惠码 2026 年初短暂上线后曾失效，建议购买前去官方确认是否有效。
